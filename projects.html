MyOwnDictionary

www.rbmk.fr
gitHub repository

MyOwnDictionary is a personal project. The goal was to give language learners a tool to create a list of unknown words and test their knowledge.
First, you have to create a "dictionary" choosing two languages. Then you add the words and their translations. And finally, you can check your dictionary words list or test yourself.
This was my first full-stack project. I built it from scratch using PHP (MVC pattern), MySql, HTML/CSS, and JavaScript. The hosting part was on an AWS EC2 server and AWS RDS for the database. I used git and bitbucket for the versioning.
The rest of the project is to rewrite it from scratch using Symfony.
