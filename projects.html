MyDictionary

www.mydictionary.org
gitHub repository

MyDictionary is a personal project. The goal was to give language learners a tool to create a list of unknown terms and test their lexicographic knowledge.
You are first encouraged to pick two languages for your glossary. Then you add terms and their translations. Using this glossary, you will be able to check your list or test yourself.
This was my first full-stack project. I built it from scratch using PHP (MVC pattern), MySql, HTML/CSS, and JavaScript. The hosting part was on an AWS EC2 server and AWS RDS was used for the database. Git and Bitbucket served for the versioning.
The remaining part of the project is to rewrite it from scratch using Symfony.
